## 应急包·速查助手 - 家庭/车载/办公 应急物资清单与演练

**本项目由阿里云ESA提供加速、计算和保护。**

> 说明：本项目用于参加“阿里云边缘安全加速ESA · Pages 边缘应用开发大赛”，提供可直接部署的应急物资盘点与演练辅助工具。

---

### 一、项目简介

`应急包·速查助手` 是一个基于 **Vue 3 + Vite** 的离线友好应用，内置家庭/车载/办公等场景的应急物资清单，支持本地勾选、备注、增删物资、导出 JSON，以及 5 步演练提示，帮助个人/家庭/团队把“知道要备”落地为“已经备好并知道怎么用”。

- 预设多场景清单：家庭 / 车载 / 办公室 / 自定义
- 本地存储，不上传数据；支持导出 JSON 备份
- 演练提示：5 步快速自测，确认物资位置与使用方法

---

### 二、功能说明

- **清单管理**
  - 按类别展示物资，勾选/取消勾选，支持备注（有效期、位置）
  - 新增/删除自定义物资
  - 本地持久化（localStorage）

- **场景切换**
  - 内置家庭、车载、办公室、自定义预设
  - 切换后可再次编辑、勾选

- **导出与重置**
  - 一键导出 JSON：包含当前场景与物资状态
  - 重置清单：回到家庭场景并重新生成清单

- **演练提示**
  - 5 步演练清单，可一键“换一组”
  - 鼓励用 10 分钟进行小演练，确保人人知道物资位置与使用方法

---

### 三、技术栈

- **前端**：Vue 3（组合式 API），Vite
- **数据存储**：localStorage（key: `emergency_kit_v1`）
- **依赖**：`nanoid` 用于本地生成物资条目 ID
- **样式**：原生 CSS，卡片式布局，响应式设计
- **部署**：适合部署在阿里云 ESA Pages（纯静态，无后端依赖）

---

### 四、本地开发与构建

```bash
cd emergency-kit
npm install
npm run dev
```

生产构建：

```bash
npm run build
```

构建产物在 `dist/`，可直接用于 ESA Pages 部署。

---

### 五、在阿里云 ESA Pages 部署

1. 将项目推送到 GitHub 公开仓库。
2. 在 ESA Pages 创建站点，选择该仓库。
3. 构建配置：
   - **构建命令**：`npm install && npm run build`
   - **输出目录**：`dist`
4. 部署完成后获取访问 URL。

### 六、作品说明（评审维度）

1. **创意卓越**
   - 把“应急物资”从“存一份清单”升级为“盘点 + 备注 + 演练”的闭环。
